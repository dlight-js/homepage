# 快速开始
DLight.js中的路由器是一个简单而强大的工具，用于管理应用程序的路由。我们提供以下特性：

* 视图定义路由
* 嵌套路由
* 动态路由
* 路由守卫
* 路由参数
* 链接
* 路由生命周期钩子
* 自定义路由信息
* View-defined routes
* Nested routes
* Dynamic routes
* Route guards
* Route parameters
* Link
* Route lifecycle hooks
* Self-defined route information

Router由@dlightjs/components包提供。你可以这样安装它：
```bash
npm install @dlightjs/components
```
